# RI7 - Treinando a Aplicação de Interfaces

Este projeto é um exercício para praticar o conceito de polimorfismo em programação orientada a objetos, focando na aplicação de interfaces para separar comportamentos.

## Objetivo

O objetivo deste exercício é treinar onde e como separar comportamentos com a aplicação de uma interface, utilizando o polimorfismo para permitir desenvolvimento flexível e reaproveitamento de código.

## Tecnologias Utilizadas

- Node.js
- Módulo readline para entrada de dados do usuário

## Como Rodar o Projeto

1. Certifique-se de que o Node.js está instalado em sua máquina.
2. Clone este repositório:
   ```
   git clone https://github.com/Raul-Germano-Rosendo/RI7.git
   ```
3. Navegue até o diretório do projeto:
   ```
   cd RI7
   ```
4. Instale as dependências (se houver):
   ```
   npm install
   ```
5. Execute o projeto:
   ```
   node index.js
   ```

   O programa irá demonstrar o polimorfismo com interfaces, mostrando diferentes animais fazendo sons. Em seguida, solicitará que você digite o nome de um animal (cachorro, gato ou vaca) para criar uma instância baseada na sua entrada.
